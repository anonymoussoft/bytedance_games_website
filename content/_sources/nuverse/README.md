## Nuverse source snapshots

This folder stores **verbatim snapshots** of publicly accessible source payloads used to seed `bytedance.games` during the MVP stage.

### What’s included
- `nuverse_block1_channel1_en-US_2026-03-18.json`
- `nuverse_block2_channel3_en-US_2026-03-18.json`
- `nuverse_block2_channel4_en-US_2026-03-18.json`

### Provenance
- The Nuverse public website (`nvsgames.com`) loads game entries via a JSON API hosted on `act.toutiaocloud.com` (see the request pattern embedded in the Nuverse site’s public JS assets).
- These snapshots are kept to support **auditing, fact-checking, and later replacement** with first-party content sources as the site matures.

### Usage rules (policy)
- Do **not** publish claims that cannot be verified from sources recorded here or other approved sources.
- Treat all images as **third-party/externally hosted** until licensing is verified; prefer linking out rather than rehosting if uncertain.
